Detailed Description

Anytime we want to implement a new command in our paint tool, we have to inherit from the command class. This forces us to implement an 'execute' and 'undo' command.

In our Draw class note that we have also explicitly marked the functions as 'override' to indicate that these functions are overriding the behavior of functions in a base class that are implemented as virtual.
